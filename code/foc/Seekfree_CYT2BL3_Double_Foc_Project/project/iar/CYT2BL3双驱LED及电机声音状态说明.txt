LED未亮起：
	单片机未运行程序

LED常亮状态：
	左右电机正常工作

LED低频慢闪：
	如果开启了失控保护，那么只要持续500ms(默认)没收到占空比控制指令，则进入保护

	FAST-FOC模式下，可能是磁编没有数据(检查磁编线)，也可能是电机堵转保护(检测三相线连接)

	BLDC模式下，电机堵转保护(检测三相线连接)

	无感电调模式下，只要两个电机没有进入闭环状态，则会低频闪烁

LED高频快闪：
	电池电压过低，不符合一般锂电池的使用范围，该闪烁的优先级比低频闪烁更高






电机声音状态：
（单击驱动板载按键、使用指令矫正零点、无感电调模式上电会使电机发出声音，声音为：嘀  嘀嘀  嘀嘀嘀，可将其理解为通过声音输出1、2、3，用于验证三相驱动能力是否正常，电机转子在不同位置时，三次声音的响度会不一样）

第一声，响一次，听到声音代表A相上桥→B相下桥回路正常工作
第二声，响两次，听到声音代表B相上桥→C相下桥回路正常工作
第三声，响三次，听到声音代表C相上桥→A相下桥回路正常工作

如果电机只响一声，说明电机线有一路是断开的或者至少有两个半桥无法正常工作，比如只响第三声，也就是连续的三个音调，说明B相接线可能断开，或者A上桥、B上桥、B下桥、C下桥中至少2个半桥无法工作

如果电机只响两声，说明驱动至少有一个半桥无法正常工作，比如只响一声和第三声，也就是单独的一个音调和连续的三个音调，说明B相上桥大概率无法正常打开
